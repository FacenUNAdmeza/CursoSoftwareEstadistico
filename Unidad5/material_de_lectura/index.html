Introducción a Python para Análisis de Datos

Material de lectura ampliado para estudiantes de grado

1. ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y con tipado dinámico. Su sintaxis limpia lo hace ideal para principiantes y expertos en análisis de datos. Creado por Guido van Rossum en 1991, Python se ha convertido en el estándar para ciencia de datos gracias a su comunidad y bibliotecas como NumPy, Pandas y Scikit-learn.

En análisis de datos, Python permite automatizar tareas, procesar grandes volúmenes de información y crear visualizaciones interactivas.

2. ¿Qué significa "Open Source"?

Python es open source, lo que significa que su código fuente es público y gratuito. Cualquiera puede contribuir, modificar o distribuirlo. Esto ha generado una comunidad global que desarrolla bibliotecas y herramientas gratuitas, acelerando la innovación en campos como IA y big data.

Ventajas: Bajo costo, actualizaciones constantes y soporte comunitario.

Nota: Proyectos como TensorFlow o Django son ejemplos de cómo el open source impulsa Python.

5. Ejemplo Práctico: Datos con Pandas

Pandas es esencial para manipular datos tabulares.

Cargar y Explorar Datos

import pandas as pd

# Crear DataFrame
datos = {
    'Nombre': ['Ana', 'Luis', 'Marta'],
    'Edad': [25, 30, 28],
    'Ciudad': ['Madrid', 'Barcelona', 'Valencia']
}
df = pd.DataFrame(datos)

# Explorar
print(df.head())
print(df.describe())
print(df['Edad'].mean())  # Media de edad

6. Visualización con Matplotlib

Matplotlib permite crear gráficos profesionales. 

import matplotlib.pyplot as plt

# Gráfico simple
x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y, marker='o')
plt.title("Gráfico de Línea Simple")
plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.show()
