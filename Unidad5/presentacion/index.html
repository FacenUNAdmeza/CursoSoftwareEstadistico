Visualización con Matplotlib

Matplotlib permite crear gráficos profesionales para visualizar datos.

# Importar la librería
import matplotlib.pyplot as plt

# Gráfico simple
x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y, marker='o')
plt.title("Gráfico de Línea Simple")
plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.show()
