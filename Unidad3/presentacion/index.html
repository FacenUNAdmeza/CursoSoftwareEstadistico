Guía de Instalación

computer Requisitos mínimos
  • Windows 10 u 11
  • Al menos 4 GB de memoria (versión MP)
  • Conexión a internet
1
Descarga: Accede al archivo de instalación mediante el link oficial proporcionado con tu licencia
2
Ejecución: Ubica el archivo descargado, haz clic derecho y selecciona "Ejecutar como administrador"
3
Asistente: Sigue las instrucciones del InstallShield Wizard, acepta el contrato de licencia
4
Información: Ingresa tu nombre, organización y selecciona "Anyone who uses this computer"
5
Edición: Elige la versión que adquiriste (Stata/MP, Stata/SE o Stata/BE)
6
Ubicación: Deja la carpeta de destino por defecto (generalmente en el disco C)
7
Activación: Al abrir el software por primera vez, activa la licencia con tu Serial number, Code y Authorization
lightbulb
Ten a mano tu información de licencia antes de iniciar la instalación

La Interfaz: Tu Cabina de Mando

Al abrir Stata, verás una interfaz dividida en varias ventanas clave. Entenderlas es vital para no perderse:

desktop_windows
Results (Centro)
Aquí aparecen los resultados de tus análisis, tablas, errores y mensajes del sistema
keyboard
Command (Abajo)
La barra donde escribes órdenes directas. Al presionar Enter, se ejecutan
history
History / Review (Izquierda)
Muestra el historial de todos los comandos que has ejecutado
data_object
Variables (Arriba derecha)
Lista todas las variables cargadas en tu base de datos actual
settings
Properties (Abajo derecha)
Muestra detalles de la variable seleccionada o de la base de datos en general
lightbulb
Puedes hacer clic en cualquier comando del historial para ejecutarlo nuevamente

¡El Secreto del Éxito! Trabaja con Do-files

warning
Regla de Oro: Aunque puedes escribir comandos directamente en la ventana principal, siempre debes trabajar en un Do-file para proyectos serios
description ¿Qué es un Do-file?
Archivo de texto simple (extensión .do) que guarda todas tus instrucciones. Te permite corregir errores, guardar tu trabajo y reproducir el análisis en el futuro
lightbulb Buenas Prácticas
comment
Comenta tu código: Usa asteriscos (*) al inicio de una línea o /* */ para bloques
folder
Organización: Usa un encabezado para describir qué hace el archivo
cleaning_services
Limpieza: Limpia la memoria al inicio con clear all
* Ejemplo de estructura de un Do-file
clear all // Limpia la memoria
cd "C:\MisDocumentos\ProyectoStata" // Define carpeta de trabajo

/* Importar datos */
sysuse auto, clear // Carga una base de ejemplo

* Análisis descriptivo
summarize price mpg
tips_and_updates
Los Do-files son tu mejor amigo, guardan tu trabajo, permiten reproducir análisis y facilitan la colaboración con otros investigadores

Manipulación y Visualización de Datos

edit
Manipulación de Datos

generate
Crea una variable nueva

replace
Modifica una variable existente

Esencial para entender códigos numéricos

Nivel Experto: Unir Bases de Datos

A veces tus datos están en dos archivos separados. ¿Cómo los unes?

vertical_align_bottom
Append (Agregar Filas)

Úsalo cuando tienes las mismas variables pero nuevas observaciones (ej. datos de enero y datos de febrero)

use datos_enero.dta, clear
append using datos_febrero.dta
check_circle
Pone una base "debajo" de la otra
check_circle
Ideal para agregar datos temporales
merge_type
Merge (Agregar Columnas)

Úsalo cuando tienes los mismos sujetos (identificados por una variable llave o ID) pero quieres agregar nuevas variables

use datos_demograficos.dta, clear
merge 1:1 id_estudiante using datos_notas.dta
info
Define la relación (1:1, 1:m, m:1)
1:1
1:m
m:1
check_circle
Stata crea la variable _merge que indica si el cruce fue exitoso
lightbulb
Después de unir bases, siempre verifica la variable _merge para asegurar que todos los datos se combinaron correctamente
