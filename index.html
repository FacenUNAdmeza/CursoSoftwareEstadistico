Unidad 1 · Introducción a los softwares estadísticos

En la práctica moderna de la estadística resulta prácticamente imprescindible utilizar un software especializado.

A diferencia de una calculadora o una hoja de cálculo básica, los softwares estadísticos están diseñados específicamente para manejar volúmenes importantes de información y para implementar procedimientos estadísticos de forma confiable.

El énfasis principal de la Unidad 1 se centra en tres ideas: (i) reconocer qué tipo de programas se consideran software estadístico, (ii) aprender a instalar y ejecutar las herramientas básicas, y (iii) identificar las partes principales de la interfaz.

Unidad 1 · Instalación básica

Instalación de los principales entornos que se utilizarán en el curso.

  1. Acceder al sitio oficial de R (CRAN) y descargar el instalador.
  2. Instalar R utilizando la configuración por defecto.
  3. Descargar e instalar RStudio Desktop, versión gratuita.
  4. Verificar que R y RStudio se inician correctamente.
  1. Acceder al sitio oficial de Anaconda y descargar el instalador.
  2. Instalar la distribución de Python aceptando las opciones recomendadas.
  3. Abrir Anaconda Navigator y lanzar JupyterLab o Jupyter Notebook.
  4. Verificar que la consola de Python y los notebooks funcionan correctamente.
  1. Verificar si la institución dispone de licencia de SPSS.
  2. En caso de utilizar PSPP o Jamovi, descargar la versión libre desde su sitio oficial.
  3. Instalar el programa y comprobar la apertura de la hoja de datos y la ventana de resultados.

Buenas prácticas iniciales

Mantener carpetas separadas para datos originales, datos procesados, scripts y resultados.

Utilizar nombres consistentes y significativos para archivos, proyectos y scripts.
