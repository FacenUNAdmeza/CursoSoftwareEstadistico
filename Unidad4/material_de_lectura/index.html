Introducción a R: Guía Básica para Estudiantes

Bienvenidos a esta guía introductoria. Aquí aprenderán qué es R, cómo funciona y cómo dar sus primeros pasos en el análisis de datos.

1. ¿Qué es R?

R es un entorno y lenguaje de programación diseñado para el análisis estadístico y la creación de gráficos. Es una herramienta poderosa que permite manipular datos, realizar cálculos y generar visualizaciones de alta calidad,.

A diferencia de programas como Excel o SPSS donde se usan muchos menús, R funciona principalmente a través de una línea de comandos, lo que le da una gran flexibilidad y potencia. R permite realizar un análisis estadístico en una serie de pasos, guardando resultados intermedios en "objetos" para ser usados posteriormente.

2. ¿Qué significa "Open Source" (Código Abierto)?

R es una implementación open source del lenguaje S. Esto implica varias ventajas clave para estudiantes y profesionales:

  • Es gratuito: A diferencia de otros softwares comerciales como S-Plus o SPSS, R se puede descargar y usar sin costo alguno.
  • Comunidad activa: Existe una comunidad global de desarrolladores y usuarios que constantemente mejoran el programa y crean nuevas herramientas (paquetes) para ampliar sus capacidades.
  • Transparencia: Al ser de código abierto, se puede examinar cómo están hechos los análisis.

3. Instalación: R y RStudio

Para trabajar cómodamente, necesitamos instalar dos cosas: el "motor" (R) y la "interfaz" (RStudio). RStudio es una máscara visual que hace que usar R sea más fácil, ordenado y fluido.

Paso 1: Instalar R

Debes descargar el paquete base desde CRAN (Comprehensive R Archive Network).

  • Windows: Navega a cran.r-project.org/bin/windows/base/ y descarga el ejecutable.
  • Mac: Navega a cran.r-project.org/bin/macosx/ y descarga el paquete correspondiente.
  • Linux (Ubuntu): Se suele instalar mediante la terminal usando el comando sudo apt-get install r-base.

Paso 2: Instalar RStudio

Una vez instalado R, descarga RStudio desde su sitio oficial. RStudio te permitirá ver tus scripts, gráficos y objetos al mismo tiempo en diferentes ventanas,.

4. Comandos Básicos y Sintaxis

R distingue entre mayúsculas y minúsculas (es case-sensitive), por lo que Variable y variable serían objetos distintos,.

Asignación de variables

Para guardar un valor en un objeto (variable), usamos el símbolo <- (una flecha apuntando a la izquierda). 

x <- 10        # Guarda el número 10 en el objeto 'x'
y <- 5         # Guarda el número 5 en el objeto 'y'
nombre <- "Ana" # Guarda texto (siempre entre comillas)

Vectores

La estructura de datos más básica es el vector. Para crear uno con varios valores, usamos la función c() (concatenar),. 

edades <- c(15, 12, 27, 55)  # Crea un vector numérico
nombres <- c("Juan", "Pedro") # Crea un vector de caracteres

Operaciones Matemáticas

Puedes usar R como una calculadora,: 

suma <- 5 + 5
multiplicacion <- 10 * 2
raiz <- sqrt(16)  # Raíz cuadrada

Ayuda

Si no sabes qué hace una función, usa el signo de interrogación antes del nombre del comando,. 

?mean   # Muestra la ayuda para la función de promedio (media)

5. Ejemplo Introductorio: Cargando y Leyendo Datos

A continuación, veremos un ejemplo sencillo de cómo cargar datos que ya vienen en R y cómo leer un archivo externo.

A. Usando datos internos (Dataset Iris)

R incluye conjuntos de datos de prueba. Uno muy famoso es iris, que contiene medidas de flores. 

# 1. Cargar los datos en el entorno
data(iris)

# 2. Ver los nombres de las variables (columnas)
names(iris) 
# Resultado: "Sepal.Length" "Sepal.Width" "Petal.Length" "Petal.Width" "Species"

# 3. Ver un resumen estadístico rápido (mínimo, media, máximo)
summary(iris)

# 4. Ver las primeras filas de la tabla
head(iris)

B. Creación y Lectura de una Tabla

Imagina que tienes datos de personas. Podemos crear una tabla pequeña (Data Frame) manualmente,. 

# Crear las variables (columnas)
genero <- c("Hombre", "Mujer", "Mujer")
edad <- c(24, 32, 28)

# Crear la tabla (Data Frame) juntando las variables
datos_personas <- data.frame(genero, edad)

# Ver la tabla en pantalla
print(datos_personas)

C. Leer datos desde un archivo

Si tienes un archivo de texto (por ejemplo, datos.txt) con encabezados, puedes cargarlo usando read.table. El resultado será un data frame,. 

# Leer una tabla desde un archivo
# header=TRUE indica que la primera línea tiene los nombres de las columnas
mi_tabla <- read.table("datos.txt", header = TRUE)
Tip: Para acceder a una columna específica dentro de tu tabla, usa el símbolo de dólar $. Por ejemplo: iris$Species accede a la columna de especies del conjunto de datos iris,.
