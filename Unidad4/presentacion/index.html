Introducción a R

Aprende a analizar datos con el lenguaje de programación estadístico más popular del mundo.

Ideal para estadística, ciencia de datos, investigación y visualización profesional.

¿Qué es R?

code
R es un entorno y lenguaje de programación especializado en análisis estadístico, manipulación de datos y gráficos de alta calidad.
terminal
Funciona principalmente por línea de comandos, lo que permite reproducibilidad total de los análisis (a diferencia de SPSS o Excel).
save
Los resultados se guardan en objetos reutilizables, facilitando análisis complejos paso a paso.
trending_up
Es el estándar en universidades y empresas para ciencia de datos y estadística avanzada.

¿Qué significa "Open Source"?

savings

Completamente gratuito

Sin licencias ni costos ocultos, a diferencia de SPSS, SAS o Stata.

groups

Comunidad global activa

Miles de paquetes gratuitos en CRAN (más de 20.000) que amplían sus funciones constantemente.

visibility

Transparencia total

Puedes ver y verificar el código de todos los métodos estadísticos.

update

Actualizaciones constantes

Nuevas técnicas estadísticas aparecen primero en R.

Instalación: R y RStudio

download Paso 1: Instalar R (el motor)

Descarga desde CRAN: https://cran.r-project.org/

  • Windows y Mac: instalador gráfico sencillo
  • Linux: sudo apt-get install r-base

desktop_windows Paso 2: Instalar RStudio (la interfaz recomendada)

Descarga gratuita desde: posit.co/download

Ventajas de RStudio: scripts, consola, gráficos y archivos en una sola ventana organizada.

str(iris)

Crear data.frame

alumnos <- data.frame(
  nombre = c("Ana","Luis"),
  edad = c(22, 25),
  nota = c(8.5, 9.2)
)
alumnos

Leer archivos

# CSV
datos <- read.csv("archivo.csv")

# Excel (con paquete readxl)
library(readxl)
datos <- read_excel("datos.xlsx")
Tip importante: Usa datos$columna para acceder a una variable específica, por ejemplo iris$Sepal.Length.
