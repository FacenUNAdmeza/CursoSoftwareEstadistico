Introducción a la Unidad

La Unidad II se centra en el Software SPSS (Statistical Package for the Social Sciences), una de las herramientas estadísticas más utilizadas en el ámbito académico, gubernamental y empresarial. Desarrollado inicialmente por Norman H. Nie en 1968, SPSS se ha consolidado como un estándar en el análisis estadístico por su interfaz intuitiva y su poderosa capacidad de procesamiento.

Enfoque de la unidad: Esta unidad tiene un carácter eminentemente práctico, enfocándose en el desarrollo de habilidades para el manejo eficiente de SPSS en el procesamiento, manipulación y análisis exploratorio de datos estadísticos.

Competencias a Desarrollar

  • RA2.1: Emplear rutinas, comandos y funciones del software SPSS en el procesamiento y análisis exploratorios de datos
  • Objetivo Específico 2: Aplicar el software estadístico SPSS utilizando rutinas, comandos y funciones para el procesamiento y el análisis exploratorio de datos

2.2. Ventanas de SPSS

SPSS organiza su funcionalidad en varias ventanas principales, cada una con un propósito específico:

Representación esquemática de la interfaz de SPSS con sus ventanas principales

Ventana del Editor de Datos (Data Editor)

Es la ventana principal donde se visualizan y editan los datos. Presenta dos vistas:

  • Vista de Datos (Data View): Muestra los valores de las variables en formato de hoja de cálculo
  • Vista de Variables (Variable View): Permite definir las propiedades de cada variable (nombre, tipo, etiqueta, valores, etc.)

Ventana del Visor de Resultados (Output Viewer)

Muestra todos los resultados de los análisis realizados, incluyendo tablas, gráficos y mensajes del sistema. Estructurado en dos paneles:

  • Panel de navegación (izquierda): Índice jerárquico de todos los elementos de salida
  • Panel de contenido (derecha): Visualización de tablas, gráficos y texto de resultados

Ventana del Editor de Sintaxis (Syntax Editor)

Permite escribir, editar y ejecutar comandos de sintaxis SPSS. Fundamental para:

  • Automatización de tareas repetitivas
  • Documentación del proceso de análisis
  • Reproducibilidad de resultados
  • Análisis complejos que no están disponibles en los menús
Ejemplo de sintaxis SPSS básica:
* Ejemplo de sintaxis para estadísticas descriptivas.
DESCRIPTIVES VARIABLES=edad ingreso años_experiencia
/STATISTICS=MEAN STDDEV MIN MAX.

Otras Ventanas Importantes

Ventana Función Principal Acceso
Chart Editor Edición y personalización de gráficos Doble clic en cualquier gráfico en el visor
Pivot Table Editor Edición de tablas pivote de resultados Doble clic en cualquier tabla en el visor
Text Output Editor Edición de texto en los resultados Doble clic en texto en el visor

2.4. Manipulación de Datos

Importación de Datos

SPSS puede importar datos desde múltiples formatos:

Formato Procedimiento Consideraciones
Excel (.xlsx, .xls) Archivo → Abrir → Datos Seleccionar hoja correcta, verificar nombres de variables
CSV/TXT Archivo → Leer datos de texto Especificar delimitador, formato decimal, codificación
Bases de datos Archivo → Abrir base de datos Requiere drivers ODBC, configuración de conexión
Otros paquetes estadísticos Archivo → Abrir → Datos SAS, Stata, R (mediante extensiones)

Definición de Variables

En la Vista de Variables, se definen las propiedades de cada variable:

Propiedades clave de variables en SPSS:
1. Nombre: Identificador único (sin espacios, máximo 64 caracteres)
2. Tipo: Numérico, cadena, fecha, moneda, etc.
3. Ancho: Número de caracteres/dígitos
4. Decimales: Número de decimales a mostrar
5. Etiqueta: Descripción extensa de la variable
6. Valores: Etiquetas para valores codificados (ej: 1="Hombre", 2="Mujer")
7. Valores perdidos: Especificación de códigos para valores faltantes
8. Columnas: Ancho de columna en la Vista de Datos
9. Alineación: Alineación del texto en celdas
10. Escala de medida: Nominal, ordinal, de intervalo/razón

Transformación de Datos

Transformaciones Básicas:

  • Calcular variable: Transformar → Calcular variable
  • Recodificar en diferentes variables: Transformar → Recodificar en diferentes variables
  • Recodificar en la misma variable: Transformar → Recodificar en la misma variable
  • Visualizar categorías (Visual Binning): Transformar → Visual Binning
Ejemplo de sintaxis para transformaciones:
* Crear nueva variable categórica a partir de edad.
RECODE edad (18 thru 29=1) (30 thru 44=2) (45 thru 59=3) (60 thru 100=4)
INTO grupo_edad.
VARIABLE LABELS grupo_edad 'Grupos de edad'.
VALUE LABELS grupo_edad 1 'Jóvenes (18-29)' 2 'Adultos jóvenes (30-44)'
3 'Adultos (45-59)' 4 'Adultos mayores (60+)'.
EXECUTE.

Manejo de Valores Perdidos

SPSS reconoce tres tipos de valores perdidos:

  1. Sistema-missing: Celdas vacías (representadas por un punto)
  2. Definidos por el usuario: Valores específicos definidos como perdidos (ej: 99, 999)
  3. Rango más un valor discreto: Para variables numéricas con rangos de valores perdidos
Importante: Los valores perdidos afectan los cálculos estadísticos. SPSS ofrece varias opciones para manejar valores perdidos en análisis (excluir por lista, excluir por parejas, etc.).

2.5. Análisis Exploratorio

El análisis exploratorio de datos (AED) en SPSS incluye estadísticas descriptivas y técnicas de visualización para comprender la estructura y características de los datos.

Estadísticas Descriptivas

Procedimientos Principales:

  • Frecuencias: Analizar → Estadísticos descriptivos → Frecuencias
  • Descriptivos: Analizar → Estadísticos descriptivos → Descriptivos
  • Explorar: Analizar → Estadísticos descriptivos → Explorar
  • Tablas personalizadas: Analizar → Tablas → Tablas personalizadas
Medidas Estadísticas Disponibles en SPSS
Tendencia Central
  • Media
  • Mediana
  • Moda
  • Suma
Dispersión
  • Desviación estándar
  • Varianza
  • Rango
  • Mínimo y máximo
  • Rango intercuartílico
Distribución
  • Asimetría (skewness)
  • Curtosis
  • Percentiles
Otros
  • Error estándar de la media
  • Intervalo de confianza
  • Pruebas de normalidad

Visualización de Datos

Gráficos Disponibles en SPSS:

Tipo de Gráfico Uso Recomendado Acceso
Histograma Distribución de variables continuas Grafícos → Gráficos antiguos → Histograma
Diagrama de cajas Comparación de grupos, detección de outliers Grafícos → Gráficos antiguos → Cajas
Gráfico de barras Variables categóricas, frecuencias Grafícos → Gráficos antiguos → Barras
Gráfico de sectores Proporciones de categorías Grafícos → Gráficos antiguos → Sectores
Dispersión Relación entre dos variables continuas Grafícos → Gráficos antiguos → Dispersión
Ejemplo de sintaxis para análisis exploratorio completo:
* Análisis exploratorio completo de variables demográficas.
EXAMINE VARIABLES=edad ingreso años_educacion
/PLOT BOXPLOT STEMLEAF HISTOGRAM NPPLOT
/COMPARE VARIABLES
/STATISTICS DESCRIPTIVES EXTREME
/CINTERVAL 95
/MISSING LISTWISE
/NOTOTAL.

Análisis de Normalidad

SPSS ofrece varias pruebas para evaluar la normalidad de las variables:

  • Prueba de Kolmogorov-Smirnov: Para muestras grandes (>50)
  • Prueba de Shapiro-Wilk: Para muestras pequeñas (<50)
  • Gráficos Q-Q: Evaluación visual de normalidad
  • Histogramas con curva normal: Comparación visual
Buenas prácticas: Siempre comience cualquier análisis con una exploración descriptiva completa de sus datos. Esto le permitirá detectar problemas, valores extremos, y comprender mejor las características de su muestra.

Ejercicios Prácticos

Ejercicio 1: Configuración inicial

Objetivo: Instalar y configurar SPSS en su equipo personal.

Tareas:

  1. Descargar e instalar IBM SPSS Statistics (versión académica)
  2. Configurar las opciones regionales (separador decimal, formato de fecha)
  3. Personalizar la interfaz (idioma, colores, tamaño de fuente)
  4. Crear un proyecto con la estructura de carpetas recomendada

Entregable: Captura de pantalla de SPSS instalado y funcionando.

Ejercicio 2: Manipulación básica de datos

Objetivo: Importar datos y definir correctamente las variables.

Tareas:

  1. Importar el archivo "encuesta_salud.sav" disponible en el aula virtual
  2. Revisar y corregir las propiedades de las variables (etiquetas, valores, etc.)
  3. Crear una nueva variable categórica a partir de una variable continua
  4. Identificar y definir valores perdidos apropiadamente

Entregable: Archivo .sav con datos corregidos y sintaxis .sps utilizada.

Ejercicio 3: Análisis exploratorio completo

Objetivo: Realizar un análisis exploratorio completo de un conjunto de datos.

Tareas:

  1. Calcular estadísticas descriptivas para todas las variables cuantitativas
  2. Generar tablas de frecuencia para las variables categóricas
  3. Crear al menos 5 gráficos diferentes apropiados para los datos
  4. Evaluar la normalidad de las distribuciones
  5. Identificar y documentar valores atípicos

Entregable: Archivo .spv con resultados y reporte en Word con interpretación.
