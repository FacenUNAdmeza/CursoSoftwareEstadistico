Manual Técnico de IBM SPSS Statistics

Guía integral para la gestión, transformación y análisis de datos

Resumen: Este informe consolida los procedimientos técnicos para la instalación, gestión de bases de datos y ejecución de análisis estadísticos (descriptivos e inferenciales) en IBM SPSS Statistics, basado en la documentación técnica y tutoriales especializados.

1. Instalación y Configuración

Requisitos y Proceso de Instalación

La instalación de IBM SPSS Statistics en sistemas Windows requiere privilegios de administrador. Se recomienda ejecutar el instalador (archivo .exe) haciendo clic derecho y seleccionando "Ejecutar como administrador". Es importante tener en cuenta que configuraciones estrictas del Control de Cuentas de Usuario (UAC) o antivirus podrían afectar el proceso.

Activación de la Licencia

Una vez instalado el software, se debe activar mediante el "Asistente de Licenciamiento":

  1. Inicie el asistente y seleccione "Licencia de usuario autorizado" (generalmente para copias únicas).
  2. Introduzca el código de autorización de 20 dígitos alfanuméricos provisto por su institución o proveedor.
  3. El sistema verificará la licencia a través de internet y confirmará la activación de los módulos.

2. Gestión de Datos y Variables

Interfaz Principal

El Editor de Datos de SPSS cuenta con dos vistas fundamentales:

  • Vista de Variables: Se utiliza para definir las propiedades de las variables, como el nombre, tipo, etiqueta, valores y nivel de medida.
  • Vista de Datos: Muestra los valores reales de los datos. Las filas representan casos (sujetos) y las columnas representan variables.

Definición de Propiedades de Variable

Propiedad Descripción
Nombre Debe ser único, comenzar con una letra o carácter especial (@, #, $) y no contener espacios.
Tipo Define la naturaleza del dato. Los más comunes son Numérico (para cálculos) y Cadena (alfanuméricos/texto).
Etiqueta Descripción extendida de la variable (hasta 256 caracteres) que aparece en los resultados.
Valores Asigna etiquetas descriptivas a códigos numéricos (ej. 1 = Masculino, 2 = Femenino). Es crucial para variables categóricas.
Medida Define la escala de medición: Nominal (categorías sin orden), Ordinal (categorías con jerarquía) o Escala (intervalo/razón).

Importación de Datos

SPSS permite abrir archivos de datos de diversos formatos sin necesidad de conversión intermedia. Para importar desde Excel:

  1. Vaya a Archivo > Abrir > Datos o Archivo > Importar datos.
  2. Seleccione el archivo Excel. Puede optar por leer los nombres de variables de la primera fila de datos.

3. Transformación y Manipulación de Datos

Recodificación de Variables

La recodificación permite modificar los valores de los datos para agrupar categorías o corregir errores. Existen dos métodos principales:

  • En las mismas variables: Reemplaza los valores originales. Es una modificación destructiva.
  • En distintas variables (Recomendado): Crea una nueva variable con los valores recodificados, preservando la original para futuras referencias. Es útil para convertir variables continuas (ej. edad) en rangos (ej. grupos de edad).

Calcular Variable

Permite generar nuevas variables basadas en transformaciones numéricas de las existentes (ej. sumas, promedios). Se accede desde Transformar > Calcular variable. Se pueden utilizar operadores aritméticos y funciones estadísticas como SUM o MEAN.

Ordenar y Filtrar Casos

Para organizar el análisis:

  • Ordenar Casos: (Datos > Ordenar casos) Organiza las filas en orden ascendente o descendente según una o más variables.
  • Seleccionar Casos (Filtrar): (Datos > Seleccionar casos) Permite analizar un subconjunto específico de datos que cumpla una condición (ej. puntuación < 10). Los casos no seleccionados pueden descartarse, eliminarse o copiarse a un nuevo conjunto de datos.

4. Análisis Estadístico Descriptivo

Nota sobre Datos Faltantes: Antes de analizar, decida cómo manejar los datos faltantes. Puede excluir casos según lista (si falta algún dato en las variables analizadas, se excluye el caso completo) o según pareja/análisis (se usa la mayor cantidad de datos posible por par de variables).

Procedimientos Clave

  • Frecuencias: Ideal para variables categóricas. Proporciona recuentos, porcentajes y gráficos (barras, sectores, histogramas). Útil para una primera inspección de los datos.
  • Descriptivos: Genera estadísticos de resumen univariados para variables cuantitativas (Media, Desviación Típica, Mínimo, Máximo, Curtosis, Asimetría) y permite guardar puntuaciones tipificadas (Z-scores).
  • Explorar: Ofrece estadísticos más detallados y gráficos para identificar valores atípicos (outliers) y comprobar supuestos de normalidad (pruebas Kolmogorov-Smirnov y Shapiro-Wilk).
  • Tablas de Contingencia (Crosstabs): Analiza la relación entre dos o más variables categóricas. Permite calcular la prueba de Chi-cuadrado para independencia y medidas de asociación.

5. Análisis Inferencial y Comparación de Medias

Pruebas T (Comparación de dos grupos)

  • Muestras Independientes: Compara las medias de dos grupos distintos (ej. tratamiento vs. control). Requiere una variable de agrupación.
  • Muestras Relacionadas (Paired): Compara las medias de dos variables en el mismo grupo (ej. pre-test y post-test).
  • Una Muestra: Contrasta si la media de una variable difiere de una constante especificada.

ANOVA de un Factor

Utilizado para contrastar la hipótesis de que las medias de tres o más grupos son iguales. Incluye pruebas Post Hoc (como Tukey o Bonferroni) para determinar exactamente qué medias difieren una vez se ha establecido que existen diferencias significativas.

6. Correlación, Regresión y Otros Análisis

Análisis Descripción y Aplicación
Correlaciones Bivariadas Calcula coeficientes como Pearson (datos paramétricos) o Spearman (datos ordinales/no paramétricos) para medir la relación lineal o de rango entre variables.
Regresión Lineal Estima los coeficientes de una ecuación lineal para predecir el valor de una variable dependiente basándose en variables independientes. Incluye diagnósticos de colinealidad y análisis de residuos.
Análisis Factorial Técnica de reducción de datos para identificar factores subyacentes que expliquen las correlaciones entre un conjunto de variables observadas. Incluye métodos de rotación como Varimax.
Análisis de Conglomerados Identifica grupos homogéneos de casos. Métodos disponibles: K-medias (para grandes conjuntos de datos), Jerárquico (para conjuntos pequeños) y Bietápico (maneja variables categóricas y continuas).
Pruebas No Paramétricas Utilizadas cuando no se cumplen los supuestos de normalidad. Incluye pruebas como U de Mann-Whitney, Wilcoxon y Kruskal-Wallis.
